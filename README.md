# 🎬 Movie Ticket Booking System

## 🚀 Tech Stack
- React.js
- Django
- MongoDB

## 📌 Features
- User Authentication (Login/Register)
- Movie Listing & Details
- Seat Selection
- Ticket Booking
- Payment Simulation
- Admin Panel

## ⚙️ Setup Instructions

### Backend
cd backend  
pip install -r requirements.txt  
python manage.py runserver  

### Frontend
cd frontend  
npm install  
npm start
